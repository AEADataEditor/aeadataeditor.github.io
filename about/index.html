    The AEA Data Editor’s mission is to design and oversee the AEA journals’ strategy for archiving and curating research data and promoting reproducible research.

    About: AEA Data Editor

    These web pages provide unofficial and developing guidance on the implementation of the American Economic Association (AEA)’s Data and Code Availability Policy.

    The AEA Data Editor defines and monitors the AEA journals’ approach to data and reproducibility. The Data and Code Availability Policy outlines specifics for authors. Data citations are an important component of the policy, as is compliance with registration requirements. Most replication packages are deposited at the AEA Data and Code Repository at ICPSR, but any deposit at a trusted repository is accepted. Following the structure of the template README is strongly suggested, and will become a required element in May 2021.

    The current Data Editor (2018-) is Lars Vilhuber (Cornell University).

    Important information

    The AEA Data and Code Repository can be found at https://www.openicpsr.org/openicpsr/search/aea/studies. Detailed deposit guidance is available. Data and code deposits by the AEA Data Editor can be found here.
