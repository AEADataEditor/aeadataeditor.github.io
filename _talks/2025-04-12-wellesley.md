## Date/Time

April 12-13, 2025 (two-day training session)

## Abstract

This is the in-person training session for the **AEA Data Editor Internship** program (second cohort). The internship is a collaborative effort between **Cornell University (AEA Data Editor), Wellesley College, Haverford College, University of Notre Dame, and University of Colorado Boulder**.

The training covers:
- Introduction to the scientific publication process and reproducibility checking
- Reproducible practices and template README files
- Data provenance and data citations
- Command line and Git version control
- The replication workflow and report writing
- Running statistical software (Stata and others)
- Hands-on work on test cases with assistance


For more information, see [Internship Summer 2025](https://aeadataeditor.github.io/projects/project6).
