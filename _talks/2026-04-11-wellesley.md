## Date/Time

April 11-12, 2026 (two-day training session)

## Abstract

This is the in-person training session for the **AEA Data Editor Internship** program (third cohort). The internship is a collaborative effort between **Cornell University (AEA Data Editor), Wellesley College, Haverford College, University of Notre Dame, University of Colorado Boulder**, and **Hamilton College**.

The training covers:
- Introduction to the scientific publication process and reproducibility checking
- Reproducible practices and template README files
- Data provenance and data citations
- Command line and Git version control
- The replication workflow and report writing
- Running statistical software (Stata and others)
- Hands-on work on test cases with assistance


For more information, see [Internship Summer 2026](https://aeadataeditor.github.io/projects/project7).


> NOTE: Participation in the training can be arranged, but applications to the internship must be done via participating colleges. If you are an interested local organizer, please contact Lars Vilhuber for further details on how to join in 2026 or later, no later than March 15, 2026.
